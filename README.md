# yohtarojs.com

My personal site — plain HTML/CSS/JS, no framework, no build step. Built to last
and to be editable by hand.

**Full instructions for everything (editing text, photos, blog posts, going live,
and getting found on Google) are in [GUIDE.md](GUIDE.md).**

Quick map of what I edit:

- `content.js` — front page text (name, intro, lived/work/school, links)
- `posts.js` — blog posts
- `photos.js` + `images/` — photography
- `index.html` — only the ASCII-portrait block
- `styles.css` — colours live at the top under `:root`

Preview locally: `python -m http.server 8000`, then open http://localhost:8000.
